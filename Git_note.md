## git 常用指令
  - git diff --cached (查看暂缓区域的代码改动)
  - git log -p (详细查看提交记录)
  - git log -pretty=oneline  (输出一行查看提交记录)
  - git config --list (查看git配置信息)
  - git config --global user.name "user" (配置git用户名)
  - git config --global user.email "email" （配置git用户的邮箱）
  - git init (创建本地仓库)
  - git add --all (添加所有)
  - git commit -m "本次提交描述信息" 
  - git commit -a -s
